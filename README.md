# Sistem TPA

Sistem TPA adalah aplikasi berbasis web yang dibangun menggunakan framework Laravel untuk membantu pengelolaan data santri, monitoring progres pembelajaran, serta manajemen pengguna.



## About Sistem TPA

Aplikasi ini dikembangkan sebagai sistem informasi untuk mendukung kegiatan administrasi dan monitoring pada Taman Pendidikan Al-Qur'an (TPA). Sistem ini memungkinkan pengelolaan data santri, pemantauan perkembangan belajar, serta pengaturan hak akses pengguna.

Fitur utama meliputi:
- Autentikasi pengguna
- Manajemen data santri
- Monitoring progres pembelajaran
- Manajemen peran pengguna
- Operasi CRUD data



## About Laravel

Laravel is a web application framework with expressive, elegant syntax. It aims to make development an enjoyable and creative experience. Laravel simplifies common tasks used in web applications, such as:

- Simple, fast routing engine
- Powerful dependency injection container
- Multiple back-ends for session and cache storage
- Expressive, intuitive database ORM (Eloquent)
- Database-agnostic schema migrations
- Robust background job processing
- Real-time event broadcasting

Laravel is accessible, powerful, and provides tools required for large, robust applications.
